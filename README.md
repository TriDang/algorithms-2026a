# Algorithms & Analysis — RMIT 2026A

## Objectives

- Understand common data structures and algorithms
- Analyze time and space complexity
- Use algorithmic problem-solving approaches to tackle computing problems

## Key topics

- Asymptotic analysis and complexity classes
- Core data structures: arrays, linked lists, stacks, queues, hash tables, trees, heaps
- Problem‑solving paradigms: brute force, divide and conquer, greedy, dynamic programming

Prerequisites: basic Python (OOP) and discrete mathematics
